To add files in your git repositories
1.git status
2.git init
3.git add (-A for all or .)
4.git commit -m "Add any massege"
5.git remote add origin (link of the repository)
6.git push origin master

To clone a repositories
7.git clone(only one time for that perticular file)

To sync local file with updated repositories
8.git fetch --prune origin
9.git reset --hard origin/master
10.git clean -f -d

11.git branch
12.git checkout -b "name of the branch"
13.git remote -v
14.git remote add upstream <link of the original repository> #to add original repo from where you have forked
15.git push origin <name of the branch in which you want to push>

16.git pull #when you want to pull the changes in your local repo from original github repo
